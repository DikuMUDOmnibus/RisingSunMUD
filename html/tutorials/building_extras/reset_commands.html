Reset Commands
The game world is populated through zone resets. Rooms can have reset commands attached to them that execute when zones are reset. To edit a room's reset commands, enter the room editor and then go to the room reset submenu: 
[guild_row02@moonhaven]
Current resets:

  E) edit entry
  N) new entry
  D) delete entry
